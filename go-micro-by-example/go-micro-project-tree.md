│  │          
│  └─proto                          - protobuf协议的go实现
│      └─user                       - 基于protobuf协议的微服务模型
│              user.micro.go        - 提供基于protobuf的micro生态的服务接口（含服务端、客户端） 由protoc-gen-micro生成
│              user.pb.go           - 微服务模型（struct）、消息体（struct）、服务接口（method） 由protoc-gen-go生成
│              user.proto           - 定义protobuf协议模型（service）、消息体（message）、接口服务（rpc）
